## Getting Started

Getting started takes just **30 seconds**:

```bash
# Install the server globally
npm install -g sui-cli-web-server

# Start the server
sui-cli-web-server

# Open your browser
open https://cli.firstmovers.io
```

That's it! The web app will automatically connect to your local server.
