## Quick Start (TL;DR)

```bash
# macOS
brew install node sui
npx sui-cli-web
# Open https://www.harriweb3.dev

# Linux
cargo install --locked sui
npx sui-cli-web

# Windows (PowerShell Admin)
choco install nodejs-lts sui
npx sui-cli-web
```
